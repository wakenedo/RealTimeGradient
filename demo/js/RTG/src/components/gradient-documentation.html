Documentation
UMD Import
If you don’t use a bundler, you can include dynamic-gradient directly via a script tag. The library will be available on window.DynamicGradient. 

  <script src="dist/dynamic-gradient.umd.js"></script>
   <script>
   constgradient = DynamicGradient.init("#hero",{
      type:"linear",
      colors: ["#ff7e5f","#feb47b"],
    });
   </script>
Basic Usage
Initialize a real-time gradient on a DOM element by calling DynamicGradient.init() with configuration options. 

          <div id="hero" style="height: 300px;"></div>
          

  import DynamicGradient from"dynamic-gradient";

  const gradient = DynamicGradient.init("#hero", {
    type: "linear",
    direction: "to right",
    colors: ["#ff7e5f", "#feb47b"],
    transitionDuration: 1500,
    schedule: [
      { time: "06:00", colors: ["#7fff00ff", "#ffb347ff", "#ff69b4ff"] },
      { time: "18:00", colors: ["#ff4500ff", "#9370dbff", "#4682b4ff"] },
      { time: "00:00", colors: ["#ffff54ff", "#ff7f50ff", "#daa520ff"] }
    ]
  });
Scheduling
You can define a schedule of time-based gradients that change automatically throughout the day. The last matching entry before or equal to the current time is applied. If the time is earlier than the first entry, the schedule wraps around to the last one. 

           const gradient = DynamicGradient.init("#hero", {
           type:"linear",
           colors:["#ff7f50ff","#6a5acdff"],
           schedule: [
              { time: "06:00", colors: [,"#ffb347ff"]},
              { time: "18:00", colors: ["#ff4500ff","#9370dbff"]},
              { time: "00:00", colors: ["#ffff54ff","#ff7f50ff"]},
            ],
          });
Effect Methods
Stop Effects: Immediately stops all running effects. 

                  gradient.stopEffects();
Trigger Effect Hue: Overlay a temporary effect by hue or custom colors. 

  gradient.triggerEffect({
    hue: "gold",
    duration: 2000,
  });
Trigger Effect Mutate: Overlay a temporary effect by mutating gradient using custom colors. 

  gradient.triggerEffect({
    applyColors: ["#9370db", "#4682b4"],
    duration: 2000,
  });
Persist Effect: Smoothly transition to a new gradient that stays applied. 

                    gradient.persistEffect(["#9370db", "#4682b4"], 2000);
Hue Presets
Some ready-to-use gradient palettes. 

  const hues = {
    sunset: ["#ff7e5f", "#feb47b"],
    ocean: ["#00c6ff", "#0072ff"],
    forest: ["#11998e", "#38ef7d"],
    fire: ["#ff512f", "#dd2476"],
    mono: ["#ffffff", "#000000"],
  };


  gradient.setGradient({ colors: hues.forest });
Gradient Text
Apply gradients to text by enabling the textClip option. 

  <h1 id="gradient-text">Gradient Text</h1>


  const gradient = DynamicGradient.init("#gradient-text", {
    colors: ["#ff7e5f", "#feb47b"],
    textClip: true,
  });
React Example
Use DynamicGradient inside a React component with useEffect. 

        import DynamicGradient from "dynamic-gradient";
        import { useEffect } from "react";

        export default function Hero() {
          useEffect(() => {
            const gradient = DynamicGradient.init("#hero", {
              colors: ["#ff7e5f", "#feb47b"],
            });
            return () => gradient.destroy();
          }, []);
          return <div id="hero" style={"{ height: 300 }"} />;
        }
Vue Example
Use onMounted and onUnmounted to manage the gradient lifecycle. 

        import DynamicGradient from "dynamic-gradient";
        import { onMounted, onUnmounted } from "vue";

        export default {
          setup() {
            let gradient;
            onMounted(() => {
              gradient = DynamicGradient.init("#hero", {
                colors: ["#ff7e5f", "#feb47b"],
              });
            });
            onUnmounted(() => gradient.destroy());
          },
        };
Angular Example
Manage the gradient in ngAfterViewInit and clean up in ngOnDestroy. 

        import { Component, AfterViewInit, OnDestroy } from '@angular/core';
        import DynamicGradient from 'dynamic-gradient';

        @Component({
          selector: 'app-hero',
          template: '<div id="hero" style="height:300px"></div>'
        })
        export class HeroComponent implements AfterViewInit, OnDestroy {
          private gradient: any;
          ngAfterViewInit() {
            this.gradient = DynamicGradient.init('#hero', { colors: ['#ff7e5f', '#feb47b'] });
          }
          ngOnDestroy() {
            this.gradient.destroy();
          }
        }
API Reference
DynamicGradient.init(target, options) Initialize a gradient on a DOM element.
Option Type Default Description
type "linear" | "radial" "linear" Defines the gradient type.
direction string — CSS gradient direction (e.g., "to right", "45deg").
colors string[] — Array of color hex values.
transitionDuration number 1000 Transition duration in milliseconds.
schedule { time: string, colors: string[] }[] — Optional array of time-based gradient schedules ("HH:MM").
textClip boolean false Applies gradient as background-clip for text.
DynamicGradient.triggerEffect(applyColors, hue, duration) Trigger effects on a gradient Initialized.
Option Type Default Description
applyColors ["#hex" , "#hex"] [""] Optional apply array of #hex to compose the gradient overlay.
hue "gold" | "silver" | "black" | "white" | "#hex" "white" Default gradient effect when no applyColors is passed.
duration number inherit from init() Set transition effect duration.
DynamicGradient.persistEffect(applyColors, duration ) Persist a gradient effect on a Initialized gradient.
Option Type Default Description
applyColors ["#hex" , "#hex"] [""] Optional apply array of #hex to compose the gradient overlay.
duration number inherit from init() Set transition effect duration.
